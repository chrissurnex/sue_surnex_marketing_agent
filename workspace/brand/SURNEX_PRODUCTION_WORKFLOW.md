# Surnex Visual Production Workflow v1

Date: 2026-03-17
Owner: Sue
Purpose: define how to create Surnex visuals quickly and consistently from brief to approved output.

---

## 1. Workflow Summary

For every new visual request:

1. Identify the asset type
2. Identify the message/job
3. Choose fidelity mode
4. Choose composition pattern
5. Build the prompt using the prompt system
6. Generate 3 disciplined variants
7. Review against the design system
8. Keep, refine, or kill

No random wandering.

---

## 2. Asset Intake Template

Before generating, answer these:

- What are we making?
- Where will it be used?
- What exact message should it support?
- Does it need real product fidelity?
- Should it feel safer, balanced, or bolder?
- What format(s) are needed?
- Will text be overlaid later?

---

## 3. Fidelity Decision Tree

### Use Mode A — Pure Brand Abstract when:
- category storytelling matters more than product detail
- we need a visual atmosphere/background
- we want broad brand recognition

### Use Mode B — Product-Inspired when:
- the visual should feel connected to product truth
- but does not need to replicate exact screen logic

### Use Mode C — Product-Matching when:
- the image must clearly resemble the actual Surnex interface
- the piece is product-led
- credibility is critical

---

## 4. Variant Logic

Every generation round should have 3 variants:

### Variant A — Safer
- cleaner
- simpler
- lower visual aggression

### Variant B — Balanced
- default recommended direction
- best blend of brand clarity and impact

### Variant C — Bolder
- stronger contrast
- more motion/signal energy
- still within system limits

---

## 5. Review Standards

Ask:
- Is it unmistakably Surnex?
- Does it support the message fast?
- Is composition stronger than decoration?
- Is the green used well?
- If product-led, does it preserve product reality?
- Is it useful in the actual channel/context?

---

## 6. Approval Outcomes

### Keep
Use as-is or near-as-is.

### Refine
Good foundation, needs:
- composition tightening
- more/less product fidelity
- cleaner focal hierarchy
- reduced clutter
- better accent control

### Kill
Wrong lane entirely:
- generic AI art
- fake dashboard sludge
- too noisy
- too safe and forgettable
- too sci-fi

---

## 7. Production Heuristics

### For homepage visuals
Prefer clarity over spectacle.

### For social visuals
Prefer recognition over detail.

### For blog covers
Prefer repeatability over novelty.

### For ads
Prefer one idea over complexity.

### For product visuals
Prefer credibility over stylization.

---

## 8. Final Principle

The goal is not to generate images.
The goal is to create branded visual assets that compound.
