# Surnex Master Image Prompts v1

Date: 2026-03-17
Purpose: define the canonical master prompts for each Surnex asset class so generation starts from stable foundations instead of reinvention.

---

## 1. Master Hero Prompt

> Create a premium homepage/product hero image for Surnex using the approved Surnex visual system: premium dark cinematic environment, near-black and charcoal base, controlled lime signal-routing energy, layered branded depth, modern operator-grade SaaS concept-art feel, redrawn product-inspired cues only, not literal screenshots, strong focal composition, premium and memorable, avoid clutter, avoid fake logos, avoid cyberpunk or crypto aesthetics.

### Add when needed
- use real Surnex logo only
- logo mark in brand lime
- SURNEX wordmark in white

## 2. Master Feature Image Prompt

> Create a premium feature image for Surnex using the approved Surnex visual system: dark cinematic product-marketing style, controlled lime signal-routing energy, clean premium composition, redrawn product-inspired cues only, one strong focal idea, A-level clarity floor, elegant and useful-looking, no logo, no text, no screenshot pasting, no clutter.

## 3. Master Banner Prompt

> Create a premium wide banner for Surnex using the approved Surnex visual system: dark cinematic atmosphere, strong but controlled lime signal motion, layered branded depth, clean wide composition, premium recognisable brand feel, correct real Surnex logo treatment only, logo mark in brand lime and wordmark in white, avoid clutter and fake logo variations.

## 4. Master Page Asset Prompt

> Create a real page-support image for Surnex using the approved Surnex visual system: premium dark cinematic product-marketing style, controlled lime signal energy, clean composition, no logo, no text, redrawn product cues only, suitable for a real website section, clean enough to sit above supporting copy, visually consistent with other assets in the same page set.

---

## 5. Master Reference-Guided Prompt Add-on

Use this whenever screenshots or page references are supplied:

> Use attached screenshots as reference only for rhythm, structure, and product cues. Redraw and reinterpret them into the Surnex visual system. Do not paste, copy, or literally reproduce screenshots. Keep the result branded, cleaner, and more premium than raw product UI.

---

## 6. Master Page-Set Consistency Add-on

Use this when generating multiple visuals for one page:

> These images belong to the same page set and must feel coordinated. Keep the same visual language, same level of polish, same lighting logic, same complexity level, and same overall style across all images.

---

## 7. Master No-Logo Rule Add-on

> Do not include any logo, wordmark, or text in the image.

---

## 8. Master Approved Banner Logo Add-on

> If branding appears, use only the real Surnex branding. The logo mark must stay brand lime and the SURNEX wordmark must be white.
